Wallet Warmup Help
What is Wallet Warmup?

Wallet warmup is a process of creating a transaction history for your wallets by executing regular small transactions. This helps to establish wallet activity history which can be beneficial in some DeFi ecosystems.

Configuration Options
  • Gas Fee (lamports): The amount of gas fee to use for each transaction (in lamports, where 1 SOL = 1,000,000,000 lamports)
  • Priority Fee (lamports): The priority fee to use for each transaction (helps transactions to be processed faster)
  • Wallet Private Keys: The private keys of the wallets to use for transactions (never shared with any server)
  • Number of tokens: How many different tokens to use in the warmup process
  • Token Addresses: The addresses of the tokens to use (default includes SOL and USDC)
  • Time Between Txs: The number of seconds to wait between transactions
  • Shuffle wallets: Whether to randomize the order of wallets for transactions
  • Randomize amounts: Whether to use random transaction amounts (within a safe range)
Simulation Mode

When simulation mode is enabled, transactions will be simulated but not actually sent to the blockchain. This is useful for testing your configuration without spending actual funds.

Security Notice

Your private keys are processed locally in your browser and are never sent to any server. They are only used to sign transactions locally before broadcasting to the blockchain. For extra security, consider using dedicated wallets for warmup purposes that don't hold significant funds.
